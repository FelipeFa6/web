MariaDB

Install the mariadb-server package in your system 

		$ pkg_add mariadb-server

Start/Enable service 

		$ rcctl enable mysqld
		$ rcctl start mysqld

Initial Setup MariaDB

Initializes the MariaDB data directory and creates the system tables: 

		$ mysql_install_db

Verify that service running and port is open 

		$ pgrep mysqld

Or you can also try the following: 

		$ ps aux | grep mysqld
		
		#Verify that TCP port 3306 is open using netstat command:
		# $ netstat -f inet -na
		# $ netstat -f inet -na | grep 3306

Secure MariaDB 

		$ mysql_secure_installation

Start using MariaDB 

		$ mysql -u root -p

Create a user with SQL syntax inside mariadb 

		CREATE USER 'username'@'localhost' IDENTIFIED BY 'some_pass';
		GRANT ALL PRIVILEGES ON dbName.* TO 'username'@'localhost';
		FLUSH PRIVILEGES;
		
		quit

Other Stuff

The configuration file of MariaDB is located under /etc/my.cnf

After any change the daemon should be restarted 

		$ rcctl restart mysqld
