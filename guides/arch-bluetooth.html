Arch Linux

Bluetooth

Install the required packages to start using your bluetooth device.

I mainly use bluetooth for audio purposes so I also install the pulseaudio bluetooth drivers. 

		$ pacman -S bluez bluez-utils pulseaudio-bluetooth

Start/Enable Bluetooth Services: 

		$ doas systemctl start bluetooth
		$ doas systemctl enable bluetooth

Start the bluetooth controller with: 

		$ bluetoothctl

Now run the following to setup bluetooth. 

		[bluetooth]# power on
		[bluetooth]# agent on
		[bluetooth]# default-agent

The next step is scanning near devices and pairing to them using the MAC address. 

		[bluetooth]# scan on
		[NEW] Device 2C:C5:46:5F:B8:29 HUAWEI FreeBuds Pro
		[bluetooth]# pair 2C:C5:46:5F:B8:29
		[bluetooth]# connect 2C:C5:46:5F:B8:29

Finally, if you want to automatically connect to this device in the future: 

		[bluetooth]# trust 2C:C5:46:5F:B8:29

Enable at boot

By default the bluetooth daemon is running but not powered on, this means that your device won't be able to connect inmediatly after startup.

To enable this feature go to /etc/bluetooth/main.conf, uncomment and set to true the following: 

		[Policy]
		AutoEnable=true
